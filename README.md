# Clip Compare

This is a static personal site for comparing short videos and collecting image notes.

## Add your files

Place your assets here:

- Videos: `assets/videos/`
  - `clip-a.mp4`
  - `clip-b.mp4`
  - Optional: `clip-c.mp4` (duplicate a card in `index.html`)
- Images: `assets/images/`
  - `clip-a.jpg` and `clip-b.jpg` for video posters (optional)
  - `still-1.jpg`, `still-2.jpg`, `still-3.jpg` for the gallery (optional)

Missing files auto-hide their cards on load.

## Quick tweak list

- Update titles and notes in `index.html`.
- Adjust colors or layout in `styles.css`.
- The shared video controls live in `script.js`.

## Deployment notes (Render)

- Static site is enough.
- Build command: none
- Publish directory: repo root
